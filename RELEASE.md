# TextViewer 릴리즈 노트

## v1.0.3 (2026-02-24)
- 상단 중앙 `페이지 이동` 오버레이 토글 UX로 변경(기본 숨김)
- 파일명은 상단 작게 표시, 상단 모서리 중앙에서 이동 패널 표시/숨김
- 페이지 입력 버그 수정으로 숫자 입력 후 이동 동작 안정화
- 브랜딩 아이콘 리디자인(성숙한 텍스트뷰어 톤)
- 릴리스 패키지: `app-release.apk` 업로드 정비
- 릴리스 버전 갱신: `versionCode=4`, `versionName=1.0.3`

## v1.0.2 (2026-02-24)
- 릴리스 APK 배포 방식 정리: 정식 서명된 `app-release.apk` 업로드 구성
- 앱 릴리스 번호 정렬: `versionCode=3`, `versionName=1.0.2`
- 기존 수정사항(이어읽기 즉시 진입, 페이지 이동, UI 정돈, 아이콘 개선) 반영

## v1.0.1 (2026-02-24)
- 이어읽기(History)에서 항목 선택 즉시 홈으로 이동해 읽기 화면이 바로 열리도록 수정
- 화면 좌우 `밝기` 라벨 투명 처리(글자는 숨김)로 UI 정돈
- 페이지 직접 이동 기능 추가(현재 페이지 표시, 입력 후 `이동` 버튼으로 점프)
- 앱 아이콘을 3D 느낌/청량 톤으로 리디자인
- 릴리스 버전 갱신: `versionCode=2`, `versionName=1.0.1`

## v1.0.0
- 텍스트 파일 열기, 히스토리 저장/복원, 기본 제스처 페이지 제어
