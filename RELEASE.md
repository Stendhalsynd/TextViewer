# TextViewer 릴리즈 노트

## v1.0.1 (2026-02-24)
- 이어읽기(History)에서 항목 선택 즉시 홈으로 이동해 읽기 화면이 바로 열리도록 수정
- 화면 좌우 `밝기` 라벨 투명 처리(글자는 숨김)로 UI 정돈
- 페이지 직접 이동 기능 추가(현재 페이지 표시, 입력 후 `이동` 버튼으로 점프)
- 앱 아이콘을 3D 느낌/청량 톤으로 리디자인
- 릴리스 버전 갱신: `versionCode=2`, `versionName=1.0.1`

## v1.0.0
- 텍스트 파일 열기, 히스토리 저장/복원, 기본 제스처 페이지 제어
