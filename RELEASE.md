## v1.0.20 (2026-02-27)
- 페이지 구간 계산 로직을 `추정 → 정밀 보정` 2단계 구조로 개편하고, 대형 텍스트에서 범위 탐색을 보간+이분 탐색으로 전환해 정지/크래시를 완화했습니다.
- 홈/읽기/히스토리/설정 화면을 슬랙 톤 UI로 전면 교체하고 핵심 상단 상태/텍스트 레이아웃 문구를 정렬했습니다.

- 릴리스 버전 갱신: `versionCode=20`, `versionName=1.0.20`

## v1.0.16 (2026-02-26)

- 페이지 계산 시작 시 화면 크기/가독성 임계치 기반으로 버킷화 캐시 키를 적용하고, 계산 호출을 최소화해 초기 계산 체감 지연 개선.
- 탭 기반 테마 토글 영역은 테스트 태그로 분리해 반복 제스처/페이지 이동 후에도 버튼 동작이 유지되도록 보강.
- `theme-toggle-button` UI 동작 안정성을 보장하는 Compose 안드로이드 테스트 추가.
- `app-release.apk` 갱신.
- 릴리스 버전 갱신: `versionCode=16`, `versionName=1.0.16`
